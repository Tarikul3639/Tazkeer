## ğŸš€ Getting Started

### Prerequisites

* Node.js **>= 18**
* npm or yarn
* Chrome / Edge / Firefox

---

### Installation

```bash
# Create Plasmo project in current folder (with Tailwind CSS)
npm create plasmo -- . --with-tailwindcss

# Install dependencies
npm install

# Run extension in development mode
npm run dev
```

Then load the extension from the generated **build/** directory.
